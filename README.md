# Install

```sh
git clone https://github.com/talon/bash-cli-template ~/.bin/
echo "PATH=PATH:$HOME/.bin/bins" >> ~/.bashrc
source ~/.bashrc
viro apply
```
> basically just put this repo somewhere and add [bins](./bins) to your `PATH`
> and then use `viro apply` to strap in
