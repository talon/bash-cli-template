# Viro
> bash your keyboard not your head

# Install

```sh
git clone https://github.com/talon/viro
VIRO_SRC="$(pwd)/viro/src" . viro/src/core.sh && viro install
```

# Try

if you have docker you can try viro with

```sh
docker run -it --rm talon/viro
```

## Dependencies
- fzf
- fd
- ag
- realpath
- bat
